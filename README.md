# Rocket Propulsion Analysis using Hydrogen Peroxide

## Project Overview
This project presents a thermochemical analysis of hydrogen peroxide (H₂O₂) as a green monopropellant for small-scale rocket thrusters. The objective was to evaluate its decomposition characteristics, heat release, and performance parameters for potential application in eco-friendly propulsion systems.

## Objectives
- Study the thermochemical decomposition of hydrogen peroxide.
- Evaluate hydrogen peroxide as a green monopropellant.
- Estimate thrust and performance parameters for small rocket engines.
- Analyze feasibility for sustainable rocket propulsion systems.

## Tools and Software Used
- MATLAB
- OpenRocket
- Thermodynamics and combustion analysis

## Methodology
1. Thermochemical modelling of hydrogen peroxide decomposition was performed.
2. Reaction kinetics and heat release were analyzed using MATLAB.
3. Rocket performance parameters were estimated.
4. Simulations were conducted to evaluate propulsion feasibility.

## Key Results
- Determined heat release and decomposition characteristics of H₂O₂.
- Estimated performance parameters for small thrusters.
- Identified safe operating conditions for monopropellant rocket systems.
